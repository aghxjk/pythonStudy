# 内置对象类型
## 核心数据类型

|对象类型|例子|
|:--- |:--- |
| 数字    | 1234, 3.1415, 3+4j, Decimal, Fraction |
| 字符串   | 'spam', "guido's" |
| 列表    | [1, [2, 'three'], 4] |
| 字典    | {'food':'spam', 'taste':'yum'} |
| 元组    | (1, 'spam', 4, 'U') | 
| 文件    | myfile = open('eggs', r) |
| 集合    | set('abc'), {'a', 'b', 'c'} |
| 其它类型 | 类型, None, 布尔型 |
| 编程单元 | 函数, 模块, 类 |
